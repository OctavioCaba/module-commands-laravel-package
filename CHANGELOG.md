# Changelog
Todas las modificaciones notables de este proyecto se documentarán en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto sigue [Semantic Versioning](https://semver.org/lang/es/).

## [0.0.1] - 2026-01-07
### Added
- Primera versión pública del paquete.
- Service Provider para integración automática con Laravel.
- Configuración básica de `composer.json` con autoload PSR-4.
- Integración de la funcionalidad para crear `controllers` de los módulos.
- Tests iniciales para validar la única funcionalidad integrada.

## [0.0.2] - 2026-01-08
### Fixed
- Modules location moved from `modules/` to `app/Modules/`
- Removed unused dependencies
